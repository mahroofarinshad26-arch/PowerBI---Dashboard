# Global Superstore Sales Dashboard

## Project Overview
This project presents an interactive Sales Dashboard built using Microsoft Power BI.
The dashboard provides insights into sales performance across different countries, regions, states, and cities.


## Data Cleaning
Before creating the dashboard in Microsoft Power BI, the dataset was cleaned to ensure accuracy and consistency.
Removed missing and duplicate values
Corrected data types (date, numbers, text)
Renamed columns for better understanding
Checked and fixed errors in data
Organized data for easy analysis
This step helped in creating accurate and reliable visualizations.


## Objectives

Analyze overall sales and profit
Identify top-performing regions and cities
Understand shipping trends
Provide interactive filtering for better decision-making
